# Peer_To_Peer_TrafficLights_Arduinos

2 Arduinos connceted together:
One as Master with a Pedestrian button, And the other as slave
with Traffic lights.
Connected peer to peer with analog pins using the Lib <Wire.h> 
and sending I2C.
